# Installation Steps



1. tekan tombol `windows + X`
2. tekan huruf `A`
3. copy paste `Set-ExecutionPolicy RemoteSigned` ke shell
4. tekan `enter`
5. tutup powershell
6. buka browser dan paste url ini di browser `https://github.com/rif42/cellpose0/archive/refs/heads/main.zip`
7. tunggu download selesai, lalu ekstrak file zip di sebuah folder
8. double click folder `cellpose0-main`
9. double klik file `install.ps1` atau right click `install.ps1`, lalu klik `run with PowerShell`
